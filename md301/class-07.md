## HTTP

- Who is Roy Fielding?

### s known for his pioneering work on the World Wide Web, open source, and software architecture. He wrote the standards for HTTP/1. x and URI, has been a contributor to many other Web technologies, and defined the REST architectural style

- What is the HTTP protocol that Fielding and his friends created?

### “But I did create many aspects of what we now call HTTP. I wrote most of the text that defines it, and I was the primary architect of the current version, HTTP/1.1.”

- What does a `GET` do?

### `GET` is used to request data from a specified resource. `GET` is one of the most common HTTP methods.

- What does a `POST` do?

### POST is used to send data to a server to create/update a resource.

- What does `PUT` do?

### PUT is used to send data to a server to create/update a resource.

- What does `PATCH` do?

### When a client needs to replace an existing Resource entirely, they can use PUT. When they're doing a partial update, they can use HTTP PATCH.
